Can Inspection Vision System

An advanced inspection system designed to identify small and unusual defects in canned products. The system is capable of operating on a high-speed production line, ensuring real-time detection and quality assurance. It highlights the precision and efficiency of modern industrial defect detection technologies tailored for high throughput environments.

Deploy Vision Transformer(ViT) model on C++

Implemented Vision Transformer-based image captioning in C++ using ONNX. The encoder and decoder components are separated for deployment efficiency. The model is sourced from Hugging Face, ensuring access to state-of-the-art pre-trained architectures.

Fire Detection and Calling system

Fire detection system for restaurants utilizes RTSP protocol to stream feeds from multiple CCTV cameras. Upon detecting a fire, it automatically alerts the fire brigade and police, providing the incident location for swift response. The system ensures real-time monitoring and rapid emergency coordination.
